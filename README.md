# ARTSL
Algorithm, Review, Tip, Share, Lecture

- Algorithm：每周至少做一个 leetcode 的算法题
- Review：阅读并点评至少一篇英文技术文章
- Tip：学习至少一个技术技巧
- Share：分享一篇有观点和思考的技术文章
- Lecture：学习计算机课程的一周课，我自己加的，最近要学习，替换了 Review，以后会将其加回来。

偶尔改一改表述中的逻辑，文章中格式，所以会更新过往的文章。这点我很喜欢 Git。
